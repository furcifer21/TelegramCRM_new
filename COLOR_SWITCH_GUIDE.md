# Инструкция по переключению цветовых схем

## Текущий вариант: Вариант 4 (Светлая воздушная палитра)

## Доступные варианты:

### Вариант 4: Светлая воздушная палитра (ТЕКУЩИЙ)
- **Primary**: #3b82f6 (светлый синий)
- **Text**: #475569 (очень мягкий серый)
- **Surface**: #ffffff (белые карточки)
- **Особенность**: Максимально светлый интерфейс, много белого пространства

### Вариант 5: Светлая нежная палитра
- **Primary**: #5b21b6 (нежный фиолетовый)
- **Text**: #4b5563 (мягкий серый)
- **Surface**: #fefefe (почти белый с легким теплом)
- **Особенность**: Элегантный и утонченный вид

## Как переключиться на Вариант 5:

1. Откройте файл `styles/variables.scss`
2. Замените значения цветов на значения из `styles/variables-variant5.scss`
3. Откройте файл `styles/globals.scss`
4. Замените значения CSS переменных на значения из `styles/globals-variant5.scss`

Или просто скажите мне "вариант 5" и я переключу автоматически!

## Сравнение вариантов:

| Характеристика | Вариант 4 | Вариант 5 |
|---------------|-----------|-----------|
| Яркость | Максимально светлый | Очень светлый |
| Primary цвет | Светлый синий | Нежный фиолетовый |
| Стиль | Воздушный, минималистичный | Элегантный, утонченный |
| Тени | Очень легкие | Легкие |
| Границы | Почти незаметные | Мягкие |
